## Система отступов

Система отступов используется для того, чтобы задать расстояния между элементами.
Переменные отступов не используются для задания размеров или позиционирования на странице.

Переменные хранятся в [core-components/vars/gaps.css](https://github.com/alfa-laboratory/core-components/blob/master/packages/vars/src/gaps.css).

| Название  | Значение |
| --------- | -------- |
| --gap-3xs | 2px      |
| --gap-2xs | 4px      |
| --gap-xs  | 8px      |
| --gap-s   | 12px     |
| --gap-m   | 16px     |
| --gap-l   | 20px     |
| --gap-xl  | 24px     |
| --gap-2xl | 32px     |
| --gap-3xl | 40px     |
| --gap-4xl | 48px     |
| --gap-5xl | 64px     |
| --gap-6xl | 72px     |
| --gap-7xl | 96px     |
| --gap-8xl | 128px    |

Также эти переменные доступны для отрицательных значений отступов: для этого к ним нужно добавить постфикс `*-neg`.
Например, если переменная `--gap-m` равна `16px`, то переменная `--gap-m-neg` будет равна `-16px`.

### Пример использования

```css
@import '@alfalab/core-components/vars/gaps.css';

.item {
    margin-bottom: var(--gap-xl);
}
```

### Пример использования переменной c отрицательным значением

```css
@import '@alfalab/core-components/vars/gaps.css';

.item {
    margin-bottom: var(--gap-xl-neg);
}
```
