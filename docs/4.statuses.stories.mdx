## Статусы

Статусы помогают понимать готовность компонента к внедрению.

| Стейдж | Статус         | Критерии |
| ------ | -------------- | -------- |
| 0      | Proposal       | Issue с описанием требований к новому компоненту |
| 1      | Draft          | Реализованный опубликованный компонент, не протестирован в продуктах |
| 2      | Candidate      | Протестирован в одном из продуктов, понятная документация с примерами использования, готов к внедрению |
| 3      | Recommendation | Протестирован хотя бы в двух продуктах, рекомендован к внедрению |
