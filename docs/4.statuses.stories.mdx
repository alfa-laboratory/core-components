## Статусы

Статусы помогают понимать готовность компонента к внедрению.

| Стейдж | Статус                   | Критерии                                                                                                                                                                                          |
| ------ | ------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| 0      | Proposal                 | Issue с требованиями, может соответствовать требованиям только одного продукта                                                                                                                    |
| 1      | Draft                    | Реализованный опубликованный компонент, может соответствовать требованиям только одного продукта                                                                                                  |
| 2      | Candidate Recommendation | Компонент, соответствующий требованиям ключевых продуктов, прошёл [полный процесс сбора и согласования требования](http://confluence.moscow.alfaintra.net/pages/viewpage.action?pageId=387658586) |
| 3      | Proposed Recommendation  | Компонент с хорошо описанной документацией, примерами использования, готов к внедрению                                                                                                            |
| 4      | Recommendation           | Компонент, протестированный хотя бы в двух продуктах, рекомендован к внедрению                                                                                                                    |
