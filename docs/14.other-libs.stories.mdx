## Использование с другими библиотеками

Если вы пользуетесь на проекте другими библиотеками, которые тоже используют `core-components` (например `arui-private`),
то вам желательно следить, чтобы версии `core-components` этих библиотек
совпадали с версией `core-components` на вашем проекте.

Если версии `core-components` совпадать не будут, то в общем бандле вы получите несколько версий `core-components`,
что не только влияет на размер бандла, но и может привести к неожиданным последствиям.

Чтобы использовать единую версию `core-components`, укажите ее в `package.json`:

```json
"resolutions": {
    "@alfalab/core-components": "нужная версия"
}
```
