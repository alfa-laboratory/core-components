## Описание

Набор компонентов для оформления текста, реализующий [гибридную систему типографики](https://www.figma.com/file/FWya6P55TnXkiVGaM2qBio/2020-Web-Hybrid-Typography?node-id=593%3A0)

> **_NOTE:_** Компоненты не устанавливают свойство `font-family`, кроме случаев, когда используется шрифт 'Styrene UI'. Подразумевается, что шрифт будет установлен у родителя.

## Typography.Title

<Typography.Title>Стандартный заголовок для шрифта styrene</Typography.Title>
<Typography.Title font='system'>Стандартный заголовок для шрифта system</Typography.Title>

<Story name='Typography.Title'>
    {React.createElement(() => {
        const VIEW_TYPES = [
            'xlarge',
            'large',
            'medium',
            'small',
            'xsmall',
        ];
        const color = select('color', colors, '');
        const weight = select('weight', ['regular', 'medium', 'bold'], 'medium');
        const font = select('font', ['styrene', 'system'], 'styrene');
        return VIEW_TYPES.map(view => (
            <Typography.Title view={view} color={color} weight={weight} font={font} key={view}>
                заголовок view='{view}'
            </Typography.Title>
        ))
    })}
</Story>

## Typography.Text

Компонент для оформления текста. Позволяет задать внешний вид, тэг и цвет текста.

<Story name='Typography.Text'>
    {React.createElement(() => {
        const VIEW_TYPES = [
            'primary-large',
            'primary-medium',
            'primary-small',
            'secondary-large',
            'secondary-medium',
            'secondary-small',
            'component',
            'caps',
        ];
        const color = select('color', colors, '');
        const tag = select('tag', ['div', 'p', 'span'], 'p');
        const weight = select('weight', ['regular', 'medium', 'bold'], 'regular');
        const className = text('className', '');
        return VIEW_TYPES.map(view => (
            <Typography.Text view={view} color={color} tag={tag} weight={weight} className={className} key={view}>
                {view}. Космологи́ческая сингуля́рность — состояния Вселенной в определённый момент времени в прошлом,
                когда плотность энергии (материи) и кривизна пространства-времени были очень велики — порядка
                планковских значений.
            </Typography.Text>
        ));
    })}
</Story>
