## Описание

Ссылка — элемент навигации, который обозначает возможность перехода на другую страницу
или вызова нового информационного слоя.

    </Link>

## Виды ссылок
<br />

### Обычная ссылка
Нестареющая классика. Ссылка может вести как на страницу внутри системы, так и на внешний ресурс.

<Preview>
    <Container>
        <Row>
            <Link view="primary">
                Primary
            </Link>
        </Row>
         <Row>
            <Link view="secondary">
                Secondary
            </Link>
        </Row>
         <Row>
            <Link view="default">
                Default
            </Link>
        </Row>
    </Container>
</Preview>

### Псевдоссылка
Вызывает информационный слой (collapse, dropdown, popup) без перехода на другую страницу.

<Preview>
    <Container>
        <Row>
            <Link view="primary" pseudo={true}>
                Primary
            </Link>
        </Row>
         <Row>
            <Link view="secondary" pseudo={true}>
                Secondary
            </Link>
        </Row>
         <Row>
            <Link view="default" pseudo={true}>
                Default
            </Link>
        </Row>
    </Container>
</Preview>
