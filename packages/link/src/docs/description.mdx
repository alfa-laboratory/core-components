import { Preview } from '@storybook/addon-docs/blocks';

import { Container, Row } from 'storybook/blocks/grid';

Ссылка — элемент навигации, который обозначает возможность перехода на другую страницу
или вызова нового информационного слоя.

<Preview>
    <Typography.Text view='primary-medium'>
        <Link view='default' href='https://alfabank.ru/get-money/credit-cards/100-days/'>
            Заказать карту
        </Link>
    </Typography.Text>
</Preview>



## Виды ссылок

### Обычная ссылка
Нестареющая классика. Ссылка может вести как на страницу внутри системы, так и на внешний ресурс.

<Preview>
    <Container>
        <Row>
            <Link view="primary">
                Primary
            </Link>
        </Row>
         <Row>
            <Link view="secondary">
                Secondary
            </Link>
        </Row>
         <Row>
            <Link view="default">
                Default
            </Link>
        </Row>
    </Container>
</Preview>

### Псевдоссылка
Вызывает информационный слой (collapse, dropdown, popup) без перехода на другую страницу.

<Preview>
    <Container>
        <Row>
            <Link view="primary" pseudo={true}>
                Primary
            </Link>
        </Row>
         <Row>
            <Link view="secondary" pseudo={true}>
                Secondary
            </Link>
        </Row>
         <Row>
            <Link view="default" pseudo={true}>
                Default
            </Link>
        </Row>
    </Container>
</Preview>

### Слоты для кастомизации
Добавляет слоты `leftAddons` и `rightAddons` для кастомизации ссылок.
Переданный контент будет отрисован слева или справа от текста ссылки.

<Preview>
    <Container>
        <Row>
            <Link
                view="primary"
                leftAddons={<Icon />}
            >
                Ссылка с иконкой слева
            </Link>
        </Row>
        <Row>
            <Link
                view="secondary"
                pseudo={true}
                rightAddons={<Icon />}
            >
                Ссылка с иконкой справа
            </Link>
        </Row>
    </Container>
</Preview>
