## Описание

Компонент группы чекбоксов.

<Story name='Песочница'>
    {React.createElement(() => {
        const [value, setValue] = React.useState({ one: false, two: false, three: false });
        const onChange = (_, payload) => {
            setValue({ ...value, [payload.name]: payload.checked });
        };
        const direction = select('direction', ['vertical', 'horizontal'], 'vertical');
        const error = text('error', '');
        const disabled = boolean('disabled', false);
        const label = text('label', 'Заголовок группы');
        return (
            <CheckboxGroup
                label={label}
                onChange={onChange}
                direction={direction}
                error={error}
                disabled={disabled}
            >
                <Checkbox label='Первый вариант' name='one' checked={value.one} />
                <Checkbox label='Второй вариант' name='two' checked={value.two} />
                <Checkbox label='Третий вариант' name='three' checked={value.three} />
            </CheckboxGroup>
        );
    })}
</Story>

### Группа чекбоксов, использующая компонент `Tag`

        return (

                onChange={onChange}
                direction="horizontal"

    })}
