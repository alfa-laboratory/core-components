## Форма и размеры

### SuperEllipse

128px, 80px, 64px, 48px.

```tsx live
const SIZES = ['128', '80', '64', '48'];

render(
    <Container>
        <Row>
            {SIZES.map(size => (
                <Col key={size}>
                    <SuperEllipse size={size}>
                        <SfTouchIdXxlIcon />
                    </SuperEllipse>
                </Col>
            ))}
        </Row>
    </Container>,
);
```

### Circle

const SIZES = ['80', '64', '48'];
