Компонент изображения карты.

Показывает прелоадер пока картинка не загрузится (переопределяется через переменную --card-image-background)

```tsx live
<CardImage
    cardId='EG'
    width={500}
    rounded={false}
    baseUrl='https://online.alfabank.ru/cards-images/cards/'
    className='some name'
    id='some id'
    dataTestId='test id'
/>
```
